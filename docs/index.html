This workbook contains worksheets for the eight PSYC761 sessions given by Chris Berry.

  • Session 2. 29/01/24. Multiple regression 1: multiple continuous

  • Session 3. 05/02/24. ANOVA 1

    (12/02/24. No session)

  • Session 4. 19/02/24. Multiple regression 2: one continuous, one categorical

  • Session 5. 26/02/24. Multiple regression 3: hierarchical regression

  • Session 6. 04/03/24. ANOVA 2

  • Session 7. 11/03/24. Pre-post data and clinically significant change

  • Session 8. 18/03/24. Mixed models
